# Converting a Windows XP CD Into an ISO

In this guide, a Dell Windows XP Reinstallation CD will be converted into a .ISO. The guide will also work for Retail CDs.

## Dell Windows Reinstallation CD

Insert your WIndows XP Reinstallation CD:

<img src='./images/img_001.png' alt='img_001' width='600'/>

## XP Service Pack 3

Do not integrate with the Media Centre Edition as this breaks Media Center installation:

* [Windows XP Service Pack 3](https://www.catalog.update.microsoft.com/Search.aspx?q=kb936929)

## Service Pack Integration and ISO Creation

Launch nLLite and select next:

<img src='./images/img_014.png' alt='img_014' width='600'/>

Select browse and select the WinXP folder:

<img src='./images/img_015.png' alt='img_015' width='600'/>

<img src='./images/img_016.png' alt='img_016' width='300'/>

The service pack level displays, select next:

<img src='./images/img_017.png' alt='img_017' width='600'/>

Select next:

<img src='./images/img_018.png' alt='img_018' width='600'/>

If the service pack level <3 and the edition is not Media Centre Edition, select Service Pack.

Select Bootable ISO.

Select next:

<img src='./images/img_019.png' alt='img_019' width='600'/>

Select, select:

<img src='./images/img_020.png' alt='img_020' width='600'/>

Select yes and ok:

<img src='./images/img_021.png' alt='img_021' width='300'/>

<img src='./images/img_022.png' alt='img_022' width='300'/>


Select the Windows XP Service Pack 3 Update and select Open:

<img src='./images/img_023.png' alt='img_023' width='600'/>

Select OK:

<img src='./images/img_024.png' alt='img_024' width='300'/>

The Service Pack level should be Service Pack 3. Select next:

<img src='./images/img_025.png' alt='img_025' width='600'/>

Change the label to Dell_XP_Pro_SP3 and select Make ISO:

<img src='./images/img_026.png' alt='img_026' width='600'/>

Save the ISO in Downloads as Dell_XP_Pro_SP3:

<img src='./images/img_027.png' alt='img_027' width='600'/>

Select next:

<img src='./images/img_028.png' alt='img_028' width='600'/>

Select finish:

<img src='./images/img_029.png' alt='img_029' width='600'/>

The ISO is now created:
